1 核心概念

docker主机（host）：安装了Docker程序的机器，Docker是直接安装在操作系统之上的。

docker客户端（Client）：连接Docker主机进行操作。

docker仓库（Registry）：用来保存各种打包好的软件镜像。

docker镜像（images）：软件打包好的镜像。

docker容器（Container）：镜像启动后的实例，称为一个容器。

使用Docker步骤：

  1. 安装Docker。
  2. 去Docker仓库找到需要的镜像。
  3. 使用Docker运行这个镜像，这个镜像就会生成一个容器。
  4. 对容器的操作就是对软件的操作。
